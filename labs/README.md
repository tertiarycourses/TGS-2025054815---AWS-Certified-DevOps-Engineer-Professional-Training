# AWS Certified DevOps Engineer Professional Lab Guide

This folder contains practical labs for TGS-2025054815 - AWS Certified DevOps Engineer Professional Training.

## Lab Sequence

| Lab | File | Outcome |
| --- | --- | --- |
| 1 | [Account Safety, IAM, DevOps Tooling, and Governance Baseline](lab-01-account-iam-devops-tooling-baseline.md) | Secure DevOps account baseline with IAM, audit, budgets, and governance notes. |
| 2 | [Infrastructure as Code with CloudFormation, CDK, Config, and Drift](lab-02-cloudformation-cdk-config-drift.md) | IaC design notes covering stacks, change sets, drift, StackSets, Config, and tagging. |
| 3 | [CI/CD with CodePipeline, CodeBuild, CodeDeploy, and Artifact Stores](lab-03-codepipeline-codebuild-codedeploy-cicd.md) | CI/CD pipeline design with build, test, deploy, approvals, artifacts, and rollback. |
| 4 | [Deployment Strategies for Lambda, ECS, EC2, and Elastic Beanstalk](lab-04-deployment-strategies-ecs-lambda-beanstalk.md) | Deployment strategy comparison for serverless, containers, and instance workloads. |
| 5 | [Systems Manager, Automation, Patch, Parameter Store, and Run Command](lab-05-systems-manager-automation-patch-parameter.md) | Fleet operations, configuration, automation, patch, and parameter management notes. |
| 6 | [Observability with CloudWatch, X-Ray, CloudTrail, EventBridge, and Alarms](lab-06-observability-cloudwatch-xray-eventbridge.md) | Monitoring and incident signal design using metrics, logs, traces, events, and alarms. |
| 7 | [High Availability, Backup, Disaster Recovery, and Auto Remediation](lab-07-resilience-backup-ha-disaster-recovery.md) | Resilience and DR pattern checklist with backup, scaling, routing, and remediation. |
| 8 | [Security, Compliance, Cost Optimization, and Exam Review](lab-08-security-cost-optimization-exam-review.md) | Security guardrails, cost review, compliance notes, and timed exam-style practice. |
